Insert - Create Activity (User Token Required)


Let's start using this activity's record then:

URL [POST] homolog.front.cloudradio.com.br:8080/activity/play/episode/{episodeId}

Example:

http://homolog.front.cloudradio.com.br:8080/activity/play/episode/392817920

Easy, right?

Warning

When creating an play activity you'll add 1 to the total amount of listens. For obvious purposes you can only play an episode once for each user for the total duration of the episode. If you send this activity again, before reaching the end of an episode, you'll receive an error message.

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
