Cloud Radio

Remove - Delete Activity (User Token Required)

[DELETE] /activity/{id}

The id of the activity you want to remove must be sent in the url as a query parameter

Example: If you wanna remove the activity 1 you should use

http://www.cloudradio.net/activity/1

Warning

When deleting an activity you'll subtract 1 from the total amount of it.

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
