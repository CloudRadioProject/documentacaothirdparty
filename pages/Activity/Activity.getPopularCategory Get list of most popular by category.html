Select- Get Most Popular for each Category

[GET] /popularCategory

RESPONSE

This route will bring a list (a array) of categories containing:

categoryId: String,
categoryName: String,
episodes: Array[object],

What the fields mean:

categoryId: the id of the category

categoryName: the name of the category

episodes: A list of objects like: https://github.com/CloudRadioProject/cloudradio-backend/wiki/Users.Create

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
