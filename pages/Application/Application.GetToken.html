URLs for Application

Auth - Get Token

URL : [POST] api.cloudradio.com.br:8080/get_application_token_/{user}

parameters

URL parameter {user}: the username for your application

In the body of the request you need to send:
password: String, (required)
appSecret: String, (required)

Understanding the fields:

password: the application password
appSecret: You unique key to access our API, you can ask for one sending an email to dmachado@cloudradio.net

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the token you must use

About Expiration

Each Application token is valid for up to 60 minutes, after that you need to get a new token to continue using our API. The way to get a new token is making a call to this rout again, using the same parameters
