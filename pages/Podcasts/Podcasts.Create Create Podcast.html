Cloud Radio

Insert - Create Podcast (User Token Required)

URL : [POST] /podcasts

Parameters

You must send a JSON Object containing the elements below

name: String, (Required)
description: String, (Required)
keywords: Array[String], (optional)
stations: Array[ObjectId], (Required)
episodes: ObjectId, (optional)
categories: ObjectId, (optional)
image: ObjectId, (optional)
feedUrl: String, (optional)
url: String, (optional)
language: String, (optional)
disqusToken: String, (optional)
forumName: String, (optional)
administrators: Array[ObjectId], (optional)
email: String, (optional)
phone: String, (optional)
facebook: String, (optional)
twitter: String, (optional)

Understanding the fields:

name: The main name/title of the station
description: describe your station as best as you can
keywords: An array containing the keywords for the podcast
stations: An array containing the stations this podcast belongs to
categories: An array containing the ids of the categories
image: the id of the image previously updated
feedUrl: the old feed url to import the previous podcast data
url: A unique URL for users to access your station
language: The language used in the podcast ex: EN, PT, ES, DE
disqusToken: The token used to access disqus data after validation, more info how to get the token at https://disqus.com/api/docs/auth/
forumName: The disqus forum name. Usually the podcast name associated in DISQUS
administrators: An array containing the Id's of the podcast administrators
email: The main email to contact this podcast
phone: The main phone to contact this podcast
facebook: The podcast's facebook page
twitter: The podcast's twitter account

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
