Cloud Radio

Select - Get Podcast (User Token Required)

URL : [GET] /podcasts

Parameters

You must send the parameters in the url.
Example: http://www.cloudradio.net/podcasts?name=scicast
http://www.cloudradio.net/podcasts?description=big+bang

name: String, (optional)
description: String, (optional)
keywords: Array of String, (optional)
categories: ObjectId, (optional)
url: String, (optional)
language: String, (optional)
limit: Integer, (required)

Understanding the fields:

name: The main name/title of the station
description: describe your station as best as you can
keywords: An array containing the keywords for the podcast
categories: An array containing the ids of the categories
url: A unique URL for users to access your station
language: The language used in the podcast ex: EN-US, PT-BR, ES-ES
limit: this parameter is composed of two fields separated by a "," :
[skip] : The amount of pages you wanna skip
[amount] : the number of results you wanna get in this response So, if you send limit=0,30 you will get the first 30 results for your search If you send limit = 1,30 you will get the next 30 results (from 31 to 60) If you send limit = 2,30 you will get the next 30 results (from 61 to 90) and so on

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
