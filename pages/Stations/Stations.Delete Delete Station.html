Cloud Radio

Remove - Delete Station (User Token Required)

URL : [DELETE] /stations/{id}

The id of the station you want to remove must be sent in the url as a query parameter
Example: If you wanna remove the station 1 you should use
http://www.cloudradio.net/stations/1

Warning

When deleting a station you'll also gonna delete all the podcasts and subsequent episodes linked to it

Only an owner can delete a station

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
