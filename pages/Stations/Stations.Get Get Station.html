Cloud Radio

Select - Get Station (User Token Required)

URL : [GET] /stations

Parameters

You must send the parameters in the url.

Example: http://www.cloudradio.net/stations?name=jovem+nerd http://www.cloudradio.net/stations?description=dicas+de+poker

name: String, (optional)
description: String, (optional)
categories : ObjectId, (optional)
image : ObjectId, (optional)
url: String, (optional)
users: ObjectId, (optional)
podcasts: ObjectId, (optional)
administrators: ObjectId, (optional)
populate: boolean, (optional)
limit: Integer, (required)

Understanding the fields:

name: The main name/title of the station
description: describe your station as best as you can
categories: The id's of the categories you wanna list
image: the id of the image you wanna search
url: the unique url for the station
users: The station owner id
podcasts: The id of a podcast that you wanna find out to which station it belongs
administrators: The user id's that can edit that station
populate: Send true if you want to have all the data from podcasts and users too
limit: The maximum return items you want. if you do not inform this value, the default will be considered as 20

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
