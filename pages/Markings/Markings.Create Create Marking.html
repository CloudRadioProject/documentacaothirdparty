Insert - Create Marking (User Token Required)

[POST] /markings

Parameters

You must send a JSON Object containing the elements below

name : String,
description : String,
start : String,
end : String,
onDemandMonetization: Boolean,
inlineMonetization: Boolean,
link: String, (optional)
image : ObjectId, (optional)
episodes: ObjectId, (required)

Understanding the fields:

name: The main name/title of the station
description: describe your station as best as you can
start: The start time for ths marking
end: The end time for this marking
onDemandMonetization: true if you want us to insert and advertisment here
inlineMonetization: true if you are talking about a specific product/service here
link: the link that you want to be showed at this time if the user clicks the photo
image: the id of the image previously updated you wanna show in this moment
episodes: the id of the episode this marking belongs to

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
