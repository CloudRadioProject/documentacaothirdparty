Auth - Get Token (Application Token Required)

This route is used to get a logged user token, some routes can only be used with this token.

URL : [POST] http://api.cloudradio.com.br:8080/users/login

Parameters

You must send the parameters in the url.

Example: http://api.cloudradio.com.br:8080/users/login?password=supersecurity&username=test&token=xrA854b5QuF93D43bNndzhLP8VQUaGpGujeqUbF8zFxHrzdMNrwsgcFFJ3HMCgSXRaaCyQdJTKbE4Fgm4VcWRvxK9qaP8v78vNNAQ3GwGS2gQ8JWuL43Rkrh
        

        

            

                
password: String, (required)

                
username: String, (required)

            

            

                
Understanding the fields:

            

                

                    
password: the user password

                    
username: The username/email informed at register

                

            

        

        

            
About Expiration

        

            
                Each User token is valid for up to 60 minutes, after that you need to get a new token to continue using our API. The way to get a new token is making a call to this rout again, using the same parameters. Note that if your application token expires when you need to make a new call to get a user token, you can always get a fresh one here.
            
        

        

            
Response

            

                The response will contain two fields:
                

                    
status: Containing 'success' or 'error

                    
data: Containing the error message or the user token
