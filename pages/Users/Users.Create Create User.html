Insert - Create User (Application Token Required )

URL : [POST] /users

Parameters

You must send a JSON Object containing the elements below

profile: Object{
name: String, (required
description: String
birthday: Date
nickname: String
picture: Id
sex: String
gender: Strin
facebookURL: String
twitterURL: String
googleURL: String}
location: Object
country: String
state: String
city: String}
contact: Object,
emails: Array[
email: String, (required
confirmed: Boolean}], (required
phonenumbers: Array[String]
access: Object
username: String, (required
password: String, (required)


Understanding the fields:

profile: { `this object contain all data related to the user profile
name: The name of the user
description: Give a cool description to yourself
birthday: When the user was born
nickname: Using a nickname is easier to find you
picture: your profile picture, it must be the id for the required image
sex: your sex: male/female
gender: how do you recognize yourself independently of your biological sex (you can see our list of all genders here
facebookURL: The user facebook link
twitterURL: The user twitter link
googleURL: The user google link }
location: { The user location info
country: The user’s country
state: The user’s state
city: The user’s city }
contact: { This object contain all the contact data
emails: An array containing the N posible emails for your account
email: the main e-mail to confirm exchange info between us and you
confirmed: Checks if it’s a valid email
phonenumbers: An array containing all the phone numbers }
access: { this object contain the login data
username: Username for you to login
password: Send a nice, secure password here }

#Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
