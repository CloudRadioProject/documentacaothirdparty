Cloud Radio

Select - Get User (Application Token Required)

URL : [GET] /users

Parameters

_ id: String, (optional)
contact.emails: String, (optional)
profile.name: String, (optional)
profile.description: String, (optional)
profile.nickname: String, (optional)
profile.sex: String, (optional)
profile.gender: String, (optional)
limit: ([skip]Integer, [amount]Integer), (required)

Understanding the fields:

_id: The user id you want to select
contact.emails: The email of the user you want to search
profile.name: The name that appears on the profile page of the user
profile.description: Remember that cool description? Search by it
profile.nickname: The user nick
profile.sex: The user sex: male/female
profile.gender: The user gender. You can see CloudRadio list of genders here
limit: this parameter is composed of two fields separated by a "," :
[skip] : The amount of pages you wanna skip
[amount] : the number of results you wanna get in this response So, if you send limit=30 you will get the first 30 results for your search If you send limit = 2,30 you will get the next 30 results (from 31 to 60) If you send limit = 3,30 you will get the next 30 results (from 61 to 90) and so on

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true
