Cloud Radio

Remove - Delete User (User Token Required)

URL : [DELETE] /users/{id}

The id of the user you want to remove must be sent in the url as a query parameter Example: You'll only be able to delete your own user, after that you'll be unable to login again or do anything else in the system. http://www.cloudradio.net/users/1

Response

The response will contain two fields:

status: Containing 'success' or 'error
data: Containing the error message or the data created/updated/removed even the flag true

Warning

When deleting a user you'll also gonna delete all the stations, podcasts, episodes, markings and everything else related to it. You can only delete an user with it's own token.
